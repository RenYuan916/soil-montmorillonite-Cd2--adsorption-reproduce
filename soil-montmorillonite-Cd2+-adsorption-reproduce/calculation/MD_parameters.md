# MD模拟核心参数（论文2.2节）
## 软件与模块
- 软件：Materials Studio 2017
- 模拟模块：Forcite
- 力场：ClayFF力场（黏土矿物专用力场）
## 模型构建
- 研究表面：蒙脱石(001)表面，尺寸36.10×31.45 Å²，厚度13.07 Å
- 水化层：42.15 Å，1600个水分子，密度1 g/cm³
- 真空层：80 Å（避免层间周期性相互作用）
- 阳离子交换容量（CEC）：1/3、2/3、1（模拟Na⁺被Cd²⁺不同程度取代）
## 模拟条件
- 系综：NVT系综（粒子数、体积、温度恒定）
- 温度控制：Nose方法，298 K（室温）
- 步长：1 fs
- 平衡时间：200 ps（体系能量与温度平衡）
- 采样时间：200 ps，每100 fs采集一次数据
## 核心计算指标
- 相对浓度分布：Cd²⁺/Na⁺在界面的距离分布
- 径向分布函数（RDF）：Cd²⁺与水分子O的配位关系，计算水合数
- 均方位移（MSD）：拟合计算阳离子扩散系数
- 表面复合物类型：内球表面复合物（IS，<4 Å）、外球表面复合物（OS，>4 Å）
